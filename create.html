Create your own fictional time

Here you can create your own fictional time. Please read carefully the following paragraphs before you start creating.

Currently at v2.1 this tool is limited and is only capable of creating symetric times. This means you won't be able to recreate the Mayan calendar or the Gregorian. At this moment this tool expects that all the times are always the same, eg. months have always 30 days no exceptions and so on. We will address this in later versions, but till then this tool is more suited for creating primarily sci-fi based times.

Once you create your time you will get a code. Copy it and go to the load you time page where you can use it to gain access to conversion tools and clocks.
